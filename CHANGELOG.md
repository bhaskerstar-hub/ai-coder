# Changelog

## [0.1.0] - 2026-03-29

### Added
- Chat participant (`@ai`) with streaming responses and conversation history
- Inline chat integration (Cmd+K) for in-editor code editing
- Tab autocomplete with fill-in-the-middle prompting and debounce
- Multi-provider LLM support: Ollama (local), OpenAI, Anthropic, Google
- Agent tools: read file, edit file, search code, list files, run terminal
- Workspace indexer with keyword-based context retrieval
- Automatic context injection: active file, open files, selected text
- Status bar showing active provider
- Configurable settings for all providers, models, and behavior
